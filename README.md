# ✦ 開運カレンダー

九星気学をベースにした行動提案型の開運カレンダーPWAです。単一HTMLファイル・外部依存なしで動作します。

## 機能

- **今日の運気スコア** — 本命星×日盤・月盤の五行関係＋特別日＋月相で算出
- **吉方位コンパス** — 月盤から8方位を判定（五黄殺・暗剣殺・本命位）
- **今日の暦** — 六曜（大安・仏滅など、旧暦計算による）・祝日・今日は何の日（366日分の記念日）
- **カレンダー** — 六曜・祝日・天赦日・一粒万倍日・新月/満月・運気バーを月表示
- **予定・メモ** — 日付タップで書き込み（端末内LocalStorage保存）
- **相性占い** — 五行の相生相剋から5タイプ判定＋苦手な相手の攻略アドバイス（設定でONにすると表示）
- **表示カスタマイズ** — 「今日」画面のカードを個別ON/OFF

## 使い方

https://azp777.github.io/kaiun-calendar/ をスマートフォンのブラウザで開き、「ホーム画面に追加」するとアプリとして使えます。

生年月日は端末内（LocalStorage）にのみ保存され、サーバーには送信されません。

## 技術メモ

- 旧暦（天保暦）: 朔（新月）と中気（太陽黄経）の天文計算による導出（qreki方式）
- 祝日: 内閣府ルールのアルゴリズム実装（固定日・ハッピーマンデー・春分秋分・振替休日・国民の休日、2020〜2030年対応）
- 免責: 本アプリの情報は参考用です。日盤は概算値を使用しています。
